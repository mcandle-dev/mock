# 백화점 BLE 결제 시스템 구현 보고서

## 1. 프로젝트 개요
본 프로젝트는 현대백화점의 설계 원칙인 '매장 직원 개입 최소화'를 목표로 한 BLE 기반 통합 결제 시스템의 시뮬레이션 프로그램입니다. 실시간 통신을 위해 WebSockets(Socket.io) 기술을 활용하여 VPOS(직원용)와 모바일 앱(고객용) 간의 유기적인 상호작용을 구현했습니다.

## 2. 주요 구현 기능
### VPOS 단말기 앱 (포트: 5173)
- **자동 BLE Advertise**: 상품 스캔 후 주변 고객 앱 자동 검색
- **혜택 자동 계산**: 서버 통신을 통한 최적 할인/적립 혜택 분석 및 표시
- **결제 상태 실시간 수신**: 고객의 앱 결제 완료 여부를 즉시 확인
- **오프라인 결제 시뮬레이션**: 실물 카드 결제 프로세스 지원

### 고객 멤버십 앱 (포트: 5174)
- **카드 탭 자동 스캔**: 하드웨어 접촉 없이 매장 위치 및 직원 자동 인식
- **실시간 주문 알림**: VPOS에서 전송된 주문 정보를 즉시 푸시 알림으로 수신
- **앱 간 결제 연동**: 카카오페이 외부 앱 연동 및 지문 인증 시뮬레이션
- **전자 영수증**: 결제 완료 후 앱 내 자동 영수증 발행 및 포인트 적립

## 3. 기술 스택 및 환경
- **언어**: JavaScript (ES6+)
- **프레임워크**: React (Vite 기반), Express.js
- **통신**: Socket.io (실시간 양방향 이벤트 통신)
- **스타일링**: Vanilla CSS (Premium Dark/Gold Theme)
- **아이콘 및 애니메이션**: Lucide-react, Framer Motion

## 4. 실행 가이드 (Windows 환경)
1. **서버 실행**: `backend` 폴더에서 `node index.js`
2. **VPOS 실행**: `vpos` 폴더에서 `npm run dev` (http://localhost:5173)
3. **고객 앱 실행**: `customer` 폴더에서 `npm run dev` (http://localhost:5174)
